Threads


Discord threads are in many ways similar to a
normal text channel. They resides in a channel
but functions as thire own separate chat with
special quirks and limitations.

Uses


Threads are used for many things. For example
having a dedicated space/chat for a specific
topic, only allowing some users to participate,
not having to make a channel and more.

Modes


There is multipel "modes" for threads.
  • Public - Anyone who can see it.
  • Private - Only users who are added
    can see it
  • Locked - Only users with the perms
    to can send messages in it.
  • Archived - Can not be seen in the
    channel list and can not be linked
    untill opned again.

Perms

There are multiple permissions related to threads.
Some are straight forward like "Send Messages in
Threads" is whats requierd to send messages in
threads, "Create Public/Private Threads" is two
seperate permmisions for making two diffrent forms
of threads. But not all, "Manage Threads" is less
obvius. It allows you to lock, close, delete, add
users, remove users, change thread names and see
private threads.

Owner


If you make a thread then you can still edit the
name and close the thread without needing the
"Mange Threads" permission. That only goes for
a thread you made.

Searching


Using the search feture in a thread will not
work unless you search in the channel where the
thread is locate. If a thread is private then
messages wont show up even if searched for.
